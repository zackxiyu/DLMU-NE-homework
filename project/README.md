关于编译运行：
  1.先下载mingw
  2.在mingw中输入以下命令以获取gtk
    pacman -S mingw-w64-x86_64-gtk3
  3.在mingw中输入以下命令以编译并运行程序
    make 或 make main
  4.在mingw中输入以下命令清除main.exe
    make clean

关于运行结果：
  1.基本情况：
      最上面一条各个按键提供交互
      左边显示捕获的数据报，也就是流量控制
      右边显示交互信息，数据报的概览和详细情况等等
  2.合法操作步骤：
      先选择网卡，一般选wlan比较好，其他网卡可能没什么数据流过
      然后点start开始抓包，点stop结束抓包
      不管是否结束抓包都可以选择数据包
      双击会跳出对话框选择查看模式，none是简介，其他是对应层信息
      单击选中数据包，点击send可以选择发送模式
      点clean清空右边的交互信息
  3.不合法操作步骤：/（自己探索吧）
